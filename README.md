# 🤖 AI TOKER - 智能对话与数字人助手

一个基于 **FastAPI** 后端与原生 Web 前端构建的 AI 聊天应用，集成文本/语音对话、会话管理、数字人提示词优化，后续将引入 **2D Live 形象并实现 AI 自主控制动作**。目前处于 Demo 阶段，但已具备完整的对话交互能力，可作为智能客服、桌面助手、编程辅助或情感陪伴等多场景的启动平台。

---

## ✨ 已实现功能

- 💬 **文本对话**：流畅的多轮 AI 文本交流，支持上下文记忆。
- 🎙️ **语音对话**：支持语音输入与合成回复。
- 📝 **聊天记录持久化**：对话自动保存，刷新或重启后仍可回溯。
- 🧠 **数字人提示词优化**：针对 AI 数字人角色进行提示词工程，提升回答风格与一致性。

## 🚀 路线图

- [ ] 集成 **2D Live 形象**，实现角色动态展示。
- [ ] **AI 自主控制动作**，根据对话内容自动生成表情与肢体动作。
- [ ] 多角色切换与定制化 UI。
- [ ] 后端性能优化与并发支持。

---

## 🎯 应用场景

- 🧑‍💼 **智能客服**：7×24h 自动应答，降低人力成本。
- 💻 **AI 桌面助手**：本地运行的轻量级助手，处理日常问答、翻译、总结等。
- 👨‍💻 **编程助手**：解答编程问题、辅助调试。
- ❤️ **情感支持**：陪聊、树洞，提供情绪疏导。
